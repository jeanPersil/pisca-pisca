Projeto Pisca-Pisca

Descrição do Projeto

O projeto é uma aplicação web que interage com o Firebase para controlar o estado de um LED (ligado/desligado) por meio de um botão. A interface permite ao usuário visualizar e alterar o estado do LED em tempo real. Quando a página é carregada, o estado atual do LED é sincronizado com o Firebase, e o botão exibe o texto apropriado, como "Ligar" ou "Desligar". O sistema garante que qualquer alteração no estado do LED seja refletida de maneira imediata no banco de dados e na interface do usuário.

passo a passo para montar

Passo 1: Componentes LED, resistor (330 ou 220 ohms), fios, microcontrolador (ex: Arduino), e fonte de energia. Passo 2: Montagem Conecte o anodo do LED ao pino digital do microcontrolador. Conecte o catodo ao GND do microcontrolador. Coloque o resistor entre o pino do LED e o pino de saída. Passo 3: Conectar ao Firebase Configure o Firebase e obtenha as credenciais para usar no código do microcontrolador. Passo 4: Programação Programe o microcontrolador para ler o estado do LED no Firebase e controlá-lo (ligar/desligar). Passo 5: Testar Verifique se o botão na interface altera o estado do LED no circuito corretamente.

Ao clicar no botão abaixo, será possível ligar e desligar o LED controlado pelo microcontrolador.
